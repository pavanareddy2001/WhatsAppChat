# WhatsAppChatExample
React native App 

## Libray used
* Navigation :- React-navigation 

## Code Setup
* To Clone this repo 
* `git clone https://github.com/pavanareddy2001/WhatsAppChat.git`
*  yarn install

* To run Android  -  `npx react-native run-android`

* To run IOS - `cd ios pod install cd.. npx react-native run-android`
